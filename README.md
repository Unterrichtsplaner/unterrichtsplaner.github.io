# 🍎 Unterrichtsplaner

Der digitale Unterrichtsplaner ist eine moderne, schnelle und zu 100% offline-fähige Web-Applikation für Lehrkräfte. Sie läuft direkt im Browser (ohne Datenbank-Backend) und speichert alle Noten, Anwesenheiten, Sitzpläne und Stundenpläne sicher und verschlüsselt lokal auf deinem Gerät.

## ✨ Features
*   **Stundenplan:** Übersichtliche Blöcke, Kalender-Integration und schnelle Navigation.
*   **Klassen & Fächer:** Dynamische Notenlisten, Mitarbeits-Tracking (🙋) und Hausaufgabenkontrolle (📚).
*   **Sitzplan:** Visueller Sitzplan mit One-Click-Bewertungen direkt im Unterricht.
*   **Dashboard:** Automatische Warnungen bei zu vielen Fehlzeiten, schlechten Noten oder vergessenen Hausaufgaben.
*   **100% Offline & Privat:** Deine Daten verlassen niemals dein iPad oder deinen PC.

## 🚀 Selbst Hosten & Anpassen (Open Source)

Jede Lehrkraft ist anders. Der Sportlehrer braucht einen "Sportzeug vergessen"-Button, die Kunstlehrerin eine "Material fehlt"-Spalte. Da herkömmliche Apps nicht alle Wünsche erfüllen können, ist dieser Unterrichtsplaner **Open Source**.

Du kannst diese App extrem leicht für dich selbst anpassen, **auch ohne programmieren zu können!**

### Der moderne KI-Weg (Anleitung)
Anstatt Programmieren zu lernen, lassen wir die KI die Arbeit für uns machen:

1. **Forke** dieses Repository auf GitHub auf deinen eigenen Account (oben rechts auf "Fork" klicken).
2. Lade dir den Ordner herunter oder nutze Tools wie [Cursor.com](https://cursor.com), [Antigravity](https://google.com) oder sogar ChatGPT (lade die Dateien `index.html` und `app.js` hoch).
3. **Gib der KI einen Befehl!** Zum Beispiel:
   > *"Hallo KI, das ist der Code für einen Unterrichtsplaner. Bitte füge im Popup-Menü für Schüler einen neuen Button namens 'Sportzeug vergessen' hinzu. Wenn ich darauf klicke, soll ein 👕-Symbol beim Schüler gespeichert werden."*
4. Die KI wird dir genau den Code geben, den du ersetzen musst, oder ihn in Editoren wie Cursor direkt für dich einfügen.
5. Lade deine neuen Dateien bei GitHub hoch und aktiviere unter den Repository-Einstellungen `Settings > Pages` die GitHub Pages.
6. Fertig! Deine ganz private, angepasste Version der App ist live.
